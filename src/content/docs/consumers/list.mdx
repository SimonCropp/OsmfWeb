The following Open Source Projects require a Maintenance Fee. This list may not be always be up to date so be sure to check each project.

* [Agents.AI](https://github.com/devlooped/Agents.AI) — Extensions for Microsoft.Agents.AI.
* [AntPlus](https://github.com/StephenHidem/AntPlus) — .NET class library for ANT+ wireless sensor devices.
* [CloudActors](https://github.com/devlooped/CloudActors) — Opinionated cloud-native distributed actors built on Orleans.
* [CloudStorageAccount](https://github.com/devlooped/CloudStorageAccount) — Brings back CloudStorageAccount for Azure Storage SDK v12+.
* [CredentialManager](https://github.com/devlooped/CredentialManager) — Cross-platform Git credential store as a NS2.0 library.
* [DependencyInjection](https://github.com/devlooped/DependencyInjection) — Attribute-based, compile-time service registration for .NET DI.
* [dotnet-retest](https://github.com/devlooped/dotnet-retest) — Automatically retries flaky .NET tests until they pass.
* [dotnet-trx](https://github.com/devlooped/dotnet-trx) — Render and analyze .NET test results (`.trx`) in the console.
* [Excelsior](https://github.com/Papyrine/Excelsior) — Data-driven Excel (`.xlsx`) generation and reading from .NET models.
* [Extensions.AI](https://github.com/devlooped/Extensions.AI) — Extensions for Microsoft.Extensions.AI.
* [GeoConvert](https://github.com/Papyrine/GeoConvert) — Converts maps between geospatial vector formats.
* [GitInfo](https://github.com/devlooped/GitInfo) — Git and SemVer info from MSBuild, C# and VB.
* [IconifyBundle](https://github.com/Papyrine/IconifyBundle) — Strongly-typed Iconify icons for .NET via a source generator.
* [JQSharp](https://github.com/devlooped/JQSharp) — A C# implementation of jq.
* [json-everything](https://github.com/json-everything/json-everything) — A suite of JSON tools for .NET (Schema, Path, Patch, Pointer).
* [json5](https://github.com/devlooped/json5) — JSON5 implementation for .NET on top of System.Text.Json.
* [MapBundle](https://github.com/Papyrine/MapBundle) — Bundled, offline map data for .NET as NuGet packages.
* [md](https://github.com/devlooped/md) — Minimal markdown output for AI-optimized consumption of dotnet tools.
* [Merq](https://github.com/devlooped/Merq) — In-process command and event message bus for .NET.
* [Morph](https://github.com/Papyrine/Morph) — Renders Word `.docx` or HTML to PNG images, no Word required.
* [MsOfficeDiff](https://github.com/Papyrine/MsOfficeDiff) — Global tools to diff Word documents and Excel workbooks.
* [Mvp.Xml](https://github.com/devlooped/Mvp.Xml) — XML utilities for .NET (EXSLT, XInclude, XPointer, and more).
* [Naiad](https://github.com/Papyrine/Naiad) — Renders Mermaid diagrams to SVG, no browser required.
* [NPOI](https://github.com/nissl-lab/npoi) — Read and write Excel and Word files in .NET (Apache POI port).
* [NuGetizer](https://github.com/devlooped/nugetizer) — Declarative, flexible NuGet packaging for .NET projects.
* [OpenXmlHtml](https://github.com/Papyrine/OpenXmlHtml) — Converts HTML into native OpenXml for `.xlsx` and `.docx`.
* [OpenXmlStreaming](https://github.com/Papyrine/OpenXmlStreaming) — Forward-only streaming writer for `.docx`/`.xlsx`/`.pptx`.
* [PandocNet](https://github.com/Papyrine/PandocNet) — Converts documents between formats via Pandoc.
* [Parchment](https://github.com/Papyrine/Parchment) — Word (`.docx`) generation from .NET models and Liquid templates.
* [runfile](https://github.com/devlooped/runfile) — Run C# programs from gists or remote repositories.
* [Schematron](https://github.com/devlooped/Schematron) — A Schematron ISO/IEC standard processor for .NET.
* [SmallSharp](https://github.com/devlooped/SmallSharp) — Create, edit and run multiple C# top-level programs in one project.
* [Spectre.Yaml](https://github.com/devlooped/Spectre.Yaml) — Display YAML in Spectre.Console.
* [StructId](https://github.com/devlooped/StructId) — Strongly-typed IDs via record structs and source generators.
* [TableStorage](https://github.com/devlooped/TableStorage) — Repository pattern with POCO support for Azure/Cosmos Table Storage.
* [ThisAssembly](https://github.com/devlooped/ThisAssembly) — Exposes assembly and project metadata as compile-time constants.
* [Web](https://github.com/devlooped/Web) — Query web content as XLinq (LINQ to XML) documents.
* [WhatsApp](https://github.com/devlooped/WhatsApp) — WhatsApp agents for .NET.
* [WiX Toolset](https://github.com/wixtoolset) — Builds Windows installers (MSI) from XML source.
* [xAI](https://github.com/devlooped/xAI) — .NET SDK for xAI's gRPC API, with Microsoft.Extensions.AI integration.
